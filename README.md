# Pharmora-A-pharmacy-database-management-system
The main purpose of this system is to create a structured relational database that automates pharmacy operations and improves data management.
Technologies Used: MY SQL
Key Features: Medicine Inventory: New medicines add karna, stock check karna, aur expiry date dekhna.

Sales & Billing: Customer ka bill banana aur stock mein se medicine kam karna.

Supplier Record: Medicines supply karne walon ka data rakhna.

Search Option: Kisi bhi medicine ko uske naam se search karna.
